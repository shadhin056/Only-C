To compete in practice SRM:

If you are new to Topcoder (TC):

1. Read Point #1, #2, #3(3.1, 3.2 in particular), #4 in this page.

2. Register

2. Download Java + Topcoder Arena as said in that page.

3. Login :)

Tutorials from TC and Stefan Pochmann will give you the guidance to the Topcoder format.

Note: Topcoder format might seem a bit strange to you at first but it's fun if you spend some time to get used to.

TC plugins:

Plugin will enhance the performance greatly in TC. A friend of me has written a nice tutorial in bengali. Using plugins you will be able to write codes using your favourite editor like MSVC++ or Dev-Cpp and test very easily and fast.

 

To compete in practice SRMs:

1. Login

2. 5/10 mins before 11 pm, Click Active users menu under the Main Menu.

3. If any of the member handles(given at the end of this mail) is active, right click that

handle and select Search. A box will appear. Click Yes.

4. You will move into the room in which the selected handle currenlty is.

5. See how many people are in that room (In the Who's here list box). If not too many wait for some time.

6. Don't open any problem before we start.

7. Feel free to chat with other members. Make use of the "Wisper" feature if you need.

8. If you feel you are a bit late, click summary, and see whether someone in that room has got a problem "Opened". If yes, you can start.
